Connect GitHub for live actions

Run triage, send clarifications to Claude, and post Figma replies straight from this page. Paste a GitHub token — it's saved only in this browser and sent nowhere but github.com. Do this once per device you use; on a shared computer, remove it when done.

Create one (~30 s):

  1. Open GitHub → generate a fine-grained token ↗
  2. Repository access → Only select repositories → this repo.
  3. Permissions → Actions: Read and write + Contents: Read and write.
  4. Generate, copy, paste below.

Run triage — which comments?

By default triage picks up comments since the last run (about the last 24 hours). Pick a wider range to re-scan older comments — an ad-hoc range doesn't move the daily cursor, so scheduled runs are unaffected. Dates are in Cairo time.
